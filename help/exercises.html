Exercises

This section contains a list of microcode exercises from the text, Computer Systems, J. Stanley Warford, Fourth edition, Jones and Bartlett, Publishers, 2010.

Each exercise contains a documentation section followed by one or more unit tests described by the UnitPre and UnitPost specifications. To complete an exercise click the Copy to Microcode button, and the documentation and unit test will be copied to the Microcode source pane in the application. You can then complete the exercise by writing the microcode program as described in the Microcode Mode section of this help system.
